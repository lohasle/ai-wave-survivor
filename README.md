# AI浪潮生存者 (AI Wave Survivor)

🤖💼 在AI横行的世界里，找到你不可替代的价值，活到下一轮裁员！

## 🎮 游戏简介

一款H5职场生存游戏，将打工人的焦虑转化为游戏乐趣。通过多章节剧情、回合制战斗、技能系统和多结局设计，体验在AI时代求生的黑色幽默。

## ✨ 核心特性

- **多章节深度剧情** - 4章主线+支线任务
- **回合制战斗系统** - 与AI面试官、Boss实时对决
- **技能树与成长** - 解锁提示词大师、AI驯兽师等独特技能
- **多结局分支** - 7种以上结局等你探索
- **职场梗与黑色幽默** - 融入各种真实打工场景

## 🚀 快速开始

### 前置要求
- Node.js 18+
- npm 或 yarn

### 安装运行

```bash
# 克隆项目
git clone https://github.com/lohasle/ai-wave-survivor.git
cd ai-wave-survivor

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 在线体验
访问: https://lohasle.github.io/ai-wave-survivor

## 📖 剧情大纲

### 第一章：入职即失业
- 场景：投简历 → 面试 → 入职培训 → 第一天上班
- Boss：AI面试官

### 第二章：职场三十六计
- 与AI协作、避免被抢功、有效发言

### 第三章：职业转生图鉴
- 6种职业分支：提示词艺术家、AI驯兽师、情感按摩师等

### 第四章：终极Boss —— AGI降临
- 最终对决，三选一胜利条件

## 🛠️ 技术栈

- **前端框架**: React 18 + Vite 5
- **状态管理**: React useState/useReducer
- **样式**: CSS3 + 响应式设计
- **目标平台**: H5（移动端优先）

## 📁 项目结构

```
ai-wave-survivor/
├── src/
│   ├── components/     # 可复用组件

```
