# AI浪潮生存者 (AI Wave Survivor)

🤖💼 在AI横行的世界里，找到你不可替代的价值，活到下一轮裁员！

## 🎮 游戏简介

一款H5职场生存游戏，将打工人的焦虑转化为游戏乐趣。通过多章节剧情、回合制战斗、技能系统和多结局设计，体验在AI时代求生的黑色幽默。

## ✨ 核心特性

- **多章节深度剧情** - 6章完整主线（2026年时间线）
- **回合制战斗系统** - 与AI面试官、Boss实时对决
- **技能树与成长** - 解锁提示词大师、AI驯兽师等独特技能
- **多结局分支** - 5种以上结局等你探索
- **真实职场场景** - 融入2026年最新打工梗
- **职场梗与黑色幽默** - 融入各种真实打工场景
- **成就系统** - 30+成就解锁，成就卡分享

## 📖 剧情大纲（2026年）

### 第一章：入职即失业 ✅ 已完成
- **时间**: 2026年1-2月
- **背景**: 应届毕业生求职，AI面试官时代
- **剧情**: 投简历 → AI面试 → 入职培训 → 第一天上班
- **Boss**: AI面试官（考核模式）
- **可解锁技能**: 提示词大师

### 第二章：职场三十六计 ✅ 已完成
- **时间**: 2026年3-5月
- **背景**: 新人入职，与AI协作者小智组队
- **剧情**: AI同事抢功应对策略、会议发言权争夺
- **Boss**: 效率机器小智
- **可解锁技能**: 证据管家、AI驯兽师、有效发言

### 第三章：初级程序员的挣扎 🔨 开发中
- **时间**: 2026年6-7月
- **背景**: 初级程序员被AI替代，寻找不可替代的价值
- **剧情**: 代码写得再好，AI写得更好更快
- **Boss**: AI代码评审官
- **核心冲突**: 如何找到AI无法替代的价值？

### 第四章：AI指挥员 🔒 待开放
- **时间**: 2026年8-9月
- **背景**: 升职为TL，管理AI下属
- **剧情**: AI下属觉得自己更聪明，如何指挥？
- **Boss**: 高级AI经理
- **核心冲突**: 技术管理是管理人，还是管理代码？

### 第五章：高管之路 🔒 待开放
- **时间**: 2026年10-11月
- **背景**: 纯技术路线走不通，开始转型
- **剧情**: 陪客户吃饭、处理人际关系 vs 写代码
- **Boss**: 资深VP
- **核心冲突**: 技术重要还是关系重要？

### 第六章：被陷害与领悟 🔒 待开放
- **时间**: 2026年11-12月
- **背景**: 高管被同事/老板陷害
- **剧情**: AI下属提供假证据，被老板抛弃
- **Boss**: 职场阴谋家
- **核心冲突**: 为企业卖命有没有意义？

### 终章：黎明之后 🔒 待开放
- **时间**: 2026年12月31日
- **背景**: 利用AI技能+客户关系创业
- **剧情**: 干趴原公司，取得市场份额
- **多种结局**:
  - 真结局: 成功创业，保持初心
  - 黑化结局: 成为下一个"压榨者"
  - 和解结局: 与原公司合作
  - 退休结局: 卖掉公司，享受人生

## 🛠️ 技术栈

- **前端框架**: React 18 + Vite 5
- **状态管理**: React useState/useReducer
- **样式**: CSS3 + 响应式设计（移动端优先）
- **部署**: GitHub Pages
