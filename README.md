# AI浪潮生存者 (AI Wave Survivor)

🤖💼 在AI横行的世界里，找到你不可替代的价值，活到下一轮裁员！

## 🎮 游戏简介

一款H5职场生存游戏，将打工人的焦虑转化为游戏乐趣。通过多章节剧情、回合制战斗、技能系统和多结局设计，体验在AI时代求生的黑色幽默。

## ✨ 核心特性

- **多章节深度剧情** - 6章完整主线+支线任务
- **回合制战斗系统** - 与AI面试官、Boss实时对决
- **技能树与成长** - 解锁提示词大师、AI驯兽师等独特技能
- **多结局分支** - 7种以上结局等你探索
- **职业转生系统** - 6种职业分支，各有独特能力
- **职场梗与黑色幽默** - 融入各种真实打工场景
- **成就系统** - 30+成就解锁，成就卡分享
- **实时排行榜** - 全球排名、好友排名、公司排名

## 📖 剧情大纲

- Boss：AI面试官（考核模式）

- 职业专属剧情线
- 能力差异化体验

### 第四章：AGI降临 🔒 待开放

### 第五章：人类反击战 🔒 待开放

## 🛠️ 技术栈

- **前端框架**: React 18 + Vite 5
- **状态管理**: React useState/useReducer
- **样式**: CSS3 + 响应式设计（移动端优先）
- **部署**: GitHub Pages

## 📁 项目结构

```
ai-wave-survivor/
```
