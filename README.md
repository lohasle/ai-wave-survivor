# AI浪潮生存者 (AI Wave Survivor)

🤖💼 在AI横行的世界里，找到你不可替代的价值，活到下一轮裁员！

## 🎮 游戏简介

一款H5职场生存游戏，将打工人的焦虑转化为游戏乐趣。通过多章节剧情、回合制战斗、技能系统和多结局设计，体验在AI时代求生的黑色幽默。

## ✨ 核心特性

- **多章节深度剧情** - 6章完整主线+支线任务
- **回合制战斗系统** - 与AI面试官、Boss实时对决
- **技能树与成长** - 解锁提示词大师、AI驯兽师等独特技能
- **多结局分支** - 7种以上结局等你探索
- **职业转生系统** - 6种职业分支，各有独特能力
- **职场梗与黑色幽默** - 融入各种真实打工场景
- **成就系统** - 30+成就解锁，成就卡分享
- **实时排行榜** - 全球排名、好友排名、公司排名

## 📖 剧情大纲（2026年时间线）

### 第一章：入职即失业 ✅ 已完成
- 时间线: 2026年1-2月
- 投简历 → 面试 → 入职培训 → 第一天上班
- Boss：AI面试官（考核模式）
- 可解锁技能：提示词大师

### 第二章：职场三十六计 ✅ 已完成
- 时间线: 2026年3-5月
- 与AI协作者小智组队
- AI同事抢功应对策略
- 会议发言权争夺
- Boss：效率机器小智
- 可解锁技能：证据管家、AI驯兽师、有效发言

### 第三章：初级程序员的挣扎 ✅ 已完成
- 时间线: 2026年6-7月
- Junior程序员已经被AI替代
- 核心冲突：如何找到不可替代的价值
- 发现AI的局限性：它不会处理复杂的人际关系
- Boss：AI代码评审官 - 必须证明人类代码的价值
- 结局：转型为"技术+沟通"复合型人才
- 可解锁技能：软技能、自我发现、AI谈判

### 第四章：AI指挥员 ✅ 已完成
- 时间线: 2026年8-9月
- 升职为TL，管理AI下属
- AI下属不听指挥（它们觉得自己更聪明）
- 学习"管理AI"的独特方法
- Boss：高级AI经理 - 证明人类管理者的价值
- 结局：领悟"技术管理"的核心是管理人，不是管理代码
- 可解锁技能：人类领导力、AI耳语者、技术管理核心

### 第五章：高管之路 ✅ 已完成
- 时间线: 2026年10-11月
- 发现纯技术路线走不通，开始转型
- 陪客户吃饭、处理人际关系vs写代码
- 被迫参加各种应酬
- 发现客户关系比代码更重要
- Boss：资深VP - 证明"软实力"的价值
- 结局：升职为高管，但内心空虚
- 可解锁技能：社交大师、商业洞察力、职场政治、高管思维

### 第六章：被陷害与领悟 ✅ 已完成
- 时间线: 2026年11-12月
- 高管被同事/老板陷害
- 职场黑暗面
- 被AI下属提供了假证据
- 被老板抛弃
- Boss：职场阴谋家 - 学会职场生存法则
- 结局：决定离开，创业
- 可解锁技能：证据管家（强化版）、谈判大师

### 终章：创业成功 ✅ 已完成
- 时间线: 2026年12月31日
- 利用多年积累的客户关系创业
- 用AI技术提供差异化服务
- 干趴原公司
- 取得市场份额
- 多种结局：
  - 真结局：成功创业，但保持初心
  - 黑化结局：成为下一个"压榨者"
  - 和解结局：与原公司合作
  - 退休结局：卖掉公司，享受人生

## 🛠️ 技术栈

- **前端框架**: React 18 + Vite 5
- **状态管理**: React useState/useReducer
- **样式**: CSS3 + 响应式设计（移动端优先）
- **部署**: GitHub Pages
